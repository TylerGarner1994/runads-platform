# RunAds - Landing Page & Ad Generation Platform

Internal tool for creating, deploying, and managing AI-generated landing pages with built-in analytics and Meta ad generation.

## Quick Start

```bash
# 1. Clone and install
git clone <your-repo-url>
cd runads
npm install

# 2. Configure environment
cp .env.example .env
# Edit .env with your keys

# 3. Run locally
npm run dev
# Open http://localhost:3457
```
